# 問題演習・模擬試験ツール

## 使い方

### 1. JSONファイルを用意
- `questions.json` など任意の名前でOKです。
- 形式は「問題JSONの形式」を参照してください。

### 2. ページを開く
- `index.html` をブラウザで開きます（サーバー不要）。

### 3. JSONを読み込む
- 「問題JSONファイルを選択」からJSONを選ぶと自動で読み込みます。
- 読み込み後、開始/終了範囲やランダム出題を設定して開始します。

### 4. モード
- **練習モード**: 問題を順に解き、回答するとその場で正誤・解説を表示。
- **模擬試験モード**: 指定した問題数を解き、最後にまとめて採点・解説を表示。


## 出題ロジック

1. まず「開始〜終了」の範囲で問題を絞り込みます。
2. その後、以下のルールで出題します。

- ランダムOFF:
  - 練習モード: 範囲の先頭から順に出題
  - 模擬試験モード: 範囲の先頭から問題数分だけ出題
- ランダムON:
  - 練習モード: 範囲内をシャッフルして出題
  - 模擬試験モード: 範囲内から問題数分をランダム抽出し、出題順もランダム


## 問題JSONの形式

- 配列の各要素が1問分のオブジェクトです。
- `answer` は選択肢キーの配列です（複数選択可）。
- `answerText` と `explanation` は表示用に使われます。

```json
[
  {
    "number": 1,
    "question": "問題文",
    "options": [
      { "key": "A", "text": "選択肢A" },
      { "key": "B", "text": "選択肢B" }
    ],
    "answer": ["A"],
    "answerText": "A. 選択肢A",
    "explanation": "解説本文"
  }
]
```
